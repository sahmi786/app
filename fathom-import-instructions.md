## HOW TO IMPORT INTO FATHOM

### Step 1: Log into Fathom
Go to https://app.fathomhq.com and sign in

### Step 2: Create New Company (or update existing)
- Click **"Add Company"** or go to **My Companies**
- Select **"Import from Excel"**

### Step 3: Upload the CSV File
- Choose the appropriate CSV file
- Fathom will auto-detect the format
- Review the mapping of accounts

### Step 4: Map Account Types
Fathom will ask you to confirm:
- **Income** = Revenue accounts
- **Cost of Sales** = Direct costs
- **Expense** = Operating expenses

### Step 5: Verify Data
- Check the imported periods (Jan 2026 - Dec 2028)
- Verify totals match the business plan

## TROUBLESHOOTING

**Issue: Fathom doesn't recognize the format**
- Ensure file is saved as CSV (comma-separated)
- Check that dates are in MMM-YY format

**Issue: Numbers import incorrectly**
- Remove any currency symbols
- Ensure no thousand separators (commas in numbers)

**Issue: Need to update data**
- Re-upload the full CSV file
- Fathom will replace existing data for matching periods
